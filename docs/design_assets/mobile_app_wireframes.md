# Gacha Scheduler 모바일 앱 와이어프레임 (현재 구현 기준)

현재 모바일 앱 프로젝트(`frontend_mobile/lib/screens/`)에 구현된 실제 소스 코드를 바탕으로 역설계(Reverse-engineer)한 와이어프레임입니다.

---

## 1. 메인 네비게이션 구조 (`home_screen.dart`)

모바일 앱의 가장 밑바탕이 되는 하단 탭(Bottom Navigation) 구조입니다.

```text
======================================================================
[선택된 탭의 본문 화면 (아래 2~5번 화면들이 여기에 표시됨)]
======================================================================
[하단 탭 바 (Bottom Navigation Bar)]
[ 📅 일정 ]  [ 🎰 가챠 ]  [ 🏠 홈(선택) ]  [ 📝 게시판 ]  [ 👤 내 정보 ]
======================================================================
```

---

## 2. 홈 탭 화면 (`home_tab_screen.dart`)

가장 먼저 진입하게 되는 홈 탭으로, 상단에 무한 스크롤되는 이벤트/픽업 배너 캐러셀이 있습니다.

```text
======================================================================
[ App Bar: "홈" ]
======================================================================
[ 🖼️ 무한 롤링 배너 이미지 (PageView)                                   ]
[                                                                    ]
[ (그라데이션 어두운 배경)                                               ]
[ 「에버나이트」 (얼음/기억)                                             ]
[ 2025-09-24 ~ 2025-10-15                                            ]
======================================================================
                                                                    
                       [ Gacha Scheduler 앱 타이틀 ]                    
                                                                    
======================================================================
```

---

## 3. 가챠 시뮬레이터 화면 (`gacha_screen.dart`)

가챠 배너를 선택하고 뽑기를 진행하는 화면입니다. (가장 구체적으로 구현되어 있습니다.)

```text
======================================================================
[ App Bar: "가챠" ]
======================================================================
[ 배너 선택 (Dropdown) ▾                                               ]
  - [원신] 나히다 픽업
----------------------------------------------------------------------
현재 천장(Pity): 35 / 90
[■■■■■■■■■□□□□□□□□□□□□□□□□□□□] (프로그레스 바)
----------------------------------------------------------------------
[ 뽑기 결과 그리드 (GridView) ]

 [👤나히다]  [👤무기]   [👤향릉]   [👤무기]   [👤무기]
 [ ★5 ]    [ ★3 ]    [ ★4 ]    [ ★3 ]    [ ★3 ]
 
 [👤무기]    [👤무기]   [👤무기]   [👤무기]   [👤무기]
 [ ★3 ]    [ ★3 ]    [ ★3 ]    [ ★3 ]    [ ★3 ]

----------------------------------------------------------------------
[  1회 뽑기 (Outlined)  ]        [  10회 뽑기 (Filled)  ]
======================================================================
```

## 4. 내 정보 (마이페이지) 화면 (`my_info_screen.dart`)

현재는 유저 데이터를 임시로 저장하고 불러오는 테스트용 디버그 폼 형태로 구현되어 있습니다.

```text
======================================================================
[ App Bar: "내 정보" ]
======================================================================
[ 키 (Key) 입력창 (TextField)                                          ]

[ 값 (Value) 입력창 (TextField)                                        ]

[             데이터 저장 (Elevated Button)             ]

[             데이터 조회 (Elevated Button)             ]

----------------------------------------------------------------------
조회된 키: test_key
조회된 값: test_value
======================================================================
```

## PM (기획자) 리뷰 노트
*   **구현 상태:** 가챠 시뮬레이터와 홈 탭의 배너 캐러셀은 UI가 매우 구체적으로 잘 잡혀 있습니다.
*   **보완 필요:** `board_screen.dart`(게시판)과 `schedule_screen.dart`(일정)는 껍데기만 남아 있으므로, 이전에 기획한 와이어프레임(웹 기준)을 모바일에 맞게 이식하는 추가 작업이 필요합니다.
*   **내 정보 탭:** 현재 디버그(키/값 저장) 형태이므로, 추후 '내가 쓴 글', '유저 프로필 이미지' 등을 보여주는 실제 유저 페이지로 개편되어야 합니다.
